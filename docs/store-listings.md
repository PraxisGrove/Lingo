# Lingo store listings

The product name is always **Lingo**. These descriptions accompany localized screenshots in the Chrome Web Store and Firefox Add-ons dashboards.

## English

**Short description:** Read translated webpages alongside the original text using your chosen translation service.

Lingo creates a bilingual reading view without replacing the original page. Choose your own translation service, control which sites translate automatically, and keep credentials and cached translations in your browser profile.

## 简体中文

**简短说明：** 使用你选择的翻译服务，对照阅读网页原文和译文。

Lingo 在保留原网页的同时创建双语对照阅读视图。你可以选择自己的翻译服务、控制哪些站点自动翻译，并将凭据和翻译缓存保留在浏览器配置中。

## 繁體中文

**簡短說明：** 使用你選擇的翻譯服務，對照閱讀網頁原文和譯文。

Lingo 在保留原網頁的同時建立雙語對照閱讀畫面。你可以選擇自己的翻譯服務、控制哪些站點自動翻譯，並將憑證和翻譯快取保留在瀏覽器設定檔中。

## 日本語

**短い説明：** 選択した翻訳サービスで、ウェブページの原文と訳文を並べて読めます。

Lingo は元のページを置き換えずに対訳表示を作ります。翻訳サービスと自動翻訳するサイトを自分で選び、認証情報と翻訳キャッシュをブラウザープロファイル内に保管できます。

## 한국어

**짧은 설명:** 선택한 번역 서비스로 웹페이지 원문과 번역문을 함께 읽으세요.

Lingo는 원본 페이지를 바꾸지 않고 이중 언어 읽기 화면을 만듭니다. 번역 서비스와 자동 번역 사이트를 직접 선택하고 인증 정보와 번역 캐시를 브라우저 프로필에 보관할 수 있습니다.

## Español

**Descripción breve:** Lee páginas traducidas junto al texto original con el servicio que elijas.

Lingo crea una vista bilingüe sin sustituir la página original. Elige tu servicio de traducción, controla qué sitios se traducen automáticamente y conserva las credenciales y la caché en tu perfil del navegador.

## Français

Lingo crée une vue bilingue sans remplacer la page originale. Choisissez votre service de traduction, contrôlez les sites traduits automatiquement et conservez les identifiants et le cache dans votre profil de navigateur.

Lingo erstellt eine zweisprachige Leseansicht, ohne die Originalseite zu ersetzen. Wähle deinen Übersetzungsdienst und automatisch übersetzte Websites selbst; Zugangsdaten und Cache bleiben im Browserprofil.
